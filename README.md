# Ciudad-Comarcas_JDBC
Proyecto desarrollado como parte de una actividad en 2º de Desarrollo de Aplicaciones Multiplataforma (DAM), enfocado en la gestión de entidades Ciudad y Comarca.
Se implementó utilizando MySQL como sistema de gestión de base de datos y JDBC para la conexión y manipulación de datos.
La arquitectura del proyecto sigue una estructura en capas aplicando el patrón DAO/Repository, separando claramente las responsabilidades entre la capa de conexión, modelo, repositorios y servicios, facilitando así el mantenimiento y la escalabilidad del código.
